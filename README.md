## RedStore E‑Commerce Website (Spring Boot)

RedStore is a full‑stack e‑commerce demo application built with **Spring Boot** on the backend and **HTML/CSS/JavaScript** on the frontend. It provides product browsing, cart and checkout flow, basic user accounts, order history, and address management.

### Tech stack

- **Backend**: Spring Boot (REST APIs), Spring Data JPA, Hibernate, MySQL
- **Frontend**: HTML, CSS, vanilla JavaScript
- **Build**: Maven

### Running the project

- **Backend**: import the Spring Boot project from `src/Backend/Shopping-website` into your IDE and run the main class `ShoppingWebsiteApplication`.
- **Frontend**: open the HTML files from `src/Frontend` in a browser (or serve them via a simple static server). The pages expect the backend to run at `http://localhost:8080/api`.
